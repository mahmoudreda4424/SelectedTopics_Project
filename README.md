## Project Overview
This project is designed to **assess soil quality for crop cultivation using real soil data**. It evaluates the suitability of soil based on multiple factors such as soil texture, pH, nutrient availability (NPK), and organic carbon content. The system also provides **region-specific feedback for Egypt**.

The goal is to help farmers, agronomists, and researchers make informed decisions about soil management and crop planning.
